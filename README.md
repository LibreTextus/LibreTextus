# LibreTextus

## Dependencies

You need to install *yaml-cpp* and *gtkmm*

```
sudo apt install libgtkmm-3.0-dev libyaml-cpp-dev
```

## How to Build

```bash
mkdir build
cd build

cmake ..
make

./LibreTextus
```
