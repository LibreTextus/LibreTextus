# LibreTextus

## Dependencies

You need to install *cmake* *build-essential* and *gtkmm*

```
sudo apt install libgtkmm-3.0-dev cmake build-essential
```

## How to Build

```bash
mkdir build
cd build

cmake ..
make

./LibreTextus
```
