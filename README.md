# LibreTextus

## Dependencies

You need to install *cmake*, *build-essential*, *gettext*, *boost-regex*, *rapidxml* and *gtkmm*.  

```
sudo apt install cmake libgtkmm-3.0-dev build-essential libboost-regex-dev gettext librapidxml-dev
```

## How to Build

```bash
mkdir build
cd build

cmake ..
make

./LibreTextus
```
